# SmartVitals – Elderly Care Mobile Application

## 🌐 Live Demo
👉 https://smartvitals-web-173759811029.asia-southeast1.run.app/

## 📌 Overview
*SmartVitals* is an Android-based healthcare support application designed specifically for elderly users. It combines health monitoring, emergency response, AI assistance, and medication management into one simple and accessible mobile system.

The application aims to improve elderly safety, independence, and access to healthcare services.

## ❗ Problem Statement
Many elderly users face daily healthcare challenges, including:
- Lack of real-time health monitoring
- Caregiver anxiety over elderly wellbeing
- Difficulty accessing emergency assistance quickly

## ✨ Key Features

### 🔐 Authentication & Security
- Secure user login and registration
- Personal profile management
- Update name, email, and password

### 🏠 Smart Dashboard
- Centralized navigation hub
- Simple and elderly-friendly interface
- Easy access to all major features

### 👤 Profile Management
- View and edit personal details easily

### 💊 Medication Management
- Track medication history
- Reminder-based medication support system

### 🤖 AI Chat Assistant
- Provides basic health guidance
- Delivers instant support and answers to user questions

### 🩺 Smart Health Scanner
- Scan prescriptions for easier reference
- Scan wounds or skin conditions for AI-based observation

### 📊 Health Monitoring
- Basic activity and health tracking
- Encourages a healthier daily lifestyle

### 🏥 Hospital Appointment Booking
- Appointment booking through integrated system (MySejahtera)
- Helps reduce waiting time and improve convenience

### 💊 Nearby Pharmacy Locator
- Detects nearby pharmacies using location services
- Helps users quickly find medicine access points

### 🚨 Emergency Ambulance Call
- One-tap emergency calling feature
- Provides instant access to critical help when needed
